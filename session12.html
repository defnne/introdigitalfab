The wiring and the system set-up is simple and could be followed through by https://www.arduino.cc/en/Tutorial/BuiltInExamples/Button and Nathan's website. and the code is <pre><code class="language-arduino"> const int A1A = 8; // define pin 3 for A-1A const int A1B = 9; // define pin 4 for A-1B const int buttonPin = 11; int buttonState = 0; void setup() { Serial.begin(9600); pinMode(A1A, OUTPUT); // specify these pins as outputs pinMode(A1B, OUTPUT); digitalWrite(A1A, LOW); // start with the motors off digitalWrite(A1B, LOW); pinMode(buttonPin, INPUT_PULLUP); } void loop() { // start the motor buttonState = digitalRead(buttonPin); Serial.println(buttonState); if (buttonState == HIGH){ digitalWrite(A1A, LOW); digitalWrite(A1B, LOW); delay(100); // allow the motor to run for 4 seconds } else{ // stop the motor digitalWrite(A1A, LOW); // setting both pins LOW stops the motor digitalWrite(A1B, HIGH); // redundant, but doesn't hurt delay(100); // keep the motor off for 2 seconds } } </code></pre> Which can be contsructed by using Nathan's website and the Arduino Forum as I linked. ![image of design](./ifinall.png) For the pump I used the gear pump design which can be found: https://www.thingiverse.com/thing:1696019
